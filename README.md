### Tech

stackoverflow uses a number of open source projects to work properly:

* [node.js] - evented I/O for the backend
* [Express] - fast node.js network app framework
* [jQuery] - duh
* [Vue] - Component
* [Vuex] - Central State
* [Firebase] - Realtime Database
* [Bootstrap] - Style Html
